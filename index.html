Edit Mode: select mode

Selection Mode: select mode

Editting Modes

Free
Change the view of the mesh, choose selecting modes, enter other editting modes, or perform edit operations.
Selecting (Shift)
While holding Shift click on vertices to select them
Moving (Left Mouse Button)
Move selected vertices to where you wish. Move vertices on the plane parallel to the screen, or hold Ctrl to move on the plane perpendicular to the Y-axis.

Selection Mode

Selection modes determine what vertices may be selected. Some modes don't allow vertices to be unselected. Re-choose any mode to clear selection.

Vertex (v)
Select only one vertex at a time.
Vertices (a)
Select and unselect any set of vertices.
Edge (e)
Select an edge. Once a vertex is selected, only vertices it is connected to become selectable.
Face (f)
Select a face. First two nodes which are selected must be connected with an edge. This edge determines two possible faces for selection. Once a vertex is selected which belongs to only one of them, the other vertices around that face are selected automatically.

Moving Mode

Moving mode is activated by pressing down the mouse button while not in selection mode. The vertices are dragged parallel to the screen unless Ctrl is pressed, in which case they move perpendicular to the Y-axis. The vertices move together (except with concentric selection).

Operations

Concentric Selection (o)
Select all vertices which are a) unselected and b) are connected to a selected vertex by an edge. The selection mode is automatically transferred to vertices selection mode. Each newly selected vertex is selected with a selection level of its previously selected neighbor + 1 (manually selected vertices are given level 1). When the set of vertices moves, each vertex moves the distance proportional to mouse_offset / vertex_selection_level.
Split (s)
In edge selection mode the selected edge is split at the midpoint and edges are added on both sides (unless on the boudary) of the new vertex (e.g. a triangle mesh remains a triangle mesh). In face selection mode a face must be a triangle. If so, it is split into four triangles and its three neighbors are each split into two (equivalent to calling split on each of its three edges and then rotating the first new edge inside the original triangle).
Rotate Edge (r)
In edge selection mode the edge between vertices a and b is reconnected to vertices a' and b' such that a' and b' are the next counter-clockwise vertices from a and b on the combined face which would result from removing the rotating edge.

View

Use arrow keys to rotate the mesh and PageDown/PageUp to zoom in/out.
